Data Attributes Support Test

Available Data Attributes

All these attributes can be set on input elements with data-mcdp attribute:

data-calendar: gregorian, persian, hijri
data-locale: en, fa, ar
data-format: YYYY/MM/DD, MM/DD/YYYY, DD/MM/YYYY
data-rtl: true, false
data-theme: light, dark
data-placement: bottom-start, bottom-end, top-start, top-end
data-time-picker: true, false
data-time-format: HH:mm, hh:mm A
data-show-seconds: true, false
data-use24-hour: true, false
data-stepping: 1, 5, 15, 30
data-show-today: true, false
data-show-clear: true, false
data-show-close: true, false
data-auto-close: true, false
data-hide-after-select: true, false
data-keyboard-navigation: true, false
data-today-highlight: true, false
data-allow-input: true, false
data-min-date: YYYY/MM/DD
data-max-date: YYYY/MM/DD
data-disabled-dates: comma-separated dates
data-disabled-days: comma-separated weekdays (0-6)
